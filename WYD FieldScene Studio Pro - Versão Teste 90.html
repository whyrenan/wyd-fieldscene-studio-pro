Versão Standalone

🛠️ WYD FieldScene Editor Standalone

🎯 Versão Portátil

Esta é uma versão standalone que funciona completamente no navegador, sem necessidade de instalação de Python, Flask ou qualquer outra dependência.

📋 Funcionalidades:

  • 17 Tipos de Controles: Suporte completo baseado nos editores profissionais
  • Leitura de Arquivos: Carrega arquivos .bin diretamente do computador
  • Visualização Interativa: Canvas SVG com zoom e pan
  • Sistema Hierárquico: Baseado em RootID oficial
  • Busca Inteligente: Busca por ID, tipo ou comentário (modo exato/parcial)
  • Ordenação por ID: Botões no painel de elementos para ordenar crescente ou decrescente
  • Exportação: Salva dados em formato JSON

⌨️ Atalhos:

  • F1: Isolar elemento selecionado
  • F2: Voltar
  • F3: Ir ao Pai
  • Ctrl+S: Salvar (quando implementado)
  • ↑ ↓ (sem Shift): Navegar na lista de elementos
  • Shift + ↑ ↓ ← →: Mover elemento selecionado no workspace
  • ← →: Mover elemento selecionado horizontalmente
  • Ctrl + Arrastar: Mover visualização (pan)
  • Scroll do Mouse: Zoom in/out

📁 Como Usar:

  1. Salve este arquivo HTML no seu computador
  2. Abra no seu navegador (Chrome, Firefox, Edge)
  3. Clique em "Carregar .bin" ou arraste um arquivo
  4. Explore e edite seus arquivos FieldScene!

Compatibilidade: Funciona em qualquer navegador moderno, sem instalação.
